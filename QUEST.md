# Тихий Контур — новая версия

Ты не получишь инструкцию «делай раз, делай два». Здесь есть только старт и правила.
Остальное — наблюдение, терпение, внимание к деталям.

## Старт

Запусти локальный сайт и начни с главной страницы.

Минимальный способ:

```bash
python -m http.server 8000
```

После этого открой в браузере:

```
http://localhost:8000/
```

Если не хочется поднимать сервер, можно открыть `index.html` напрямую, но некоторые
подсказки удобнее ловить через DevTools.

## Правила

- Внешний интернет не нужен.
- Никакого brute‑force.
- Подсказки не будут в лоб. Они будут рядом.
- Если зашёл в тупик — посмотри, что обычно скрывают: исходник, консоль, метаданные.

## Финал

В конце ты найдёшь короткую фразу. Запиши её — это завершение квеста.

Удачи. И помни: тишина не означает отсутствие сигнала.
